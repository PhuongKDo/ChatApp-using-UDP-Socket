• Launch Main_client.py
	◘ Client program that communicates with the server
##############################################################
Tools/Programs outside of program needed to operate:
• Python 3.X installed
##############################################################
Linux:
Steps to initiate:
	Open command line in the folder containing the "main_client" and "main_server"
>>	Type $python main_server.py
	Input the IP address and port number in the appropriate boxes

	Open another command line in folder
>>	Type $python main_client.py

	Input the same IP address and port number in to the appropriate boxes
***	IP and Port must be the same for each
